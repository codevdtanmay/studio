# **App Name**: Yoga for Her

## Core Features:

- User Authentication: Secure login using Aadhaar/Mobile number with OTP verification.
- Profile Management: Collect user details (name, age, location, etc.) and demographic information.
- Problem Selection: Allow users to select their specific problems/causes (Dysmenorrhea, Irregular Periods, etc.) and related symptoms via dropdown and checkboxes.
- Stage Selection: Categorize users into reproductive, perimenopausal, and postmenopausal stages based on age and allow manual selection.
- Personalized Yoga Modules: Dynamically display yoga modules based on selected stage, with videos, duration, frequency, and instructions.
- Progress Tracking: Allow users to track their progress and save their progress for review.
- Admin Dashboard: Enable admin/researcher access to view participant lists, filter by various parameters, download data, and upload body composition reports.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to represent calmness and femininity.
- Background color: Pale off-white (#FAFAFA), providing a clean and peaceful backdrop.
- Accent color: Light Green (#90EE90), symbolizing health and vitality, for interactive elements and calls to action.
- Body and headline font: 'Alegreya', a serif font that combines elegance with readability, for headlines and body text alike.
- Note: currently only Google Fonts are supported.
- Use minimalist, line-style icons that are intuitive and align with the theme of well-being and health.
- Employ a clean, spacious layout with clear section divisions for easy navigation and readability.
- Incorporate subtle, gentle animations (e.g., fade-in effects, smooth transitions) to enhance user experience without being distracting.
